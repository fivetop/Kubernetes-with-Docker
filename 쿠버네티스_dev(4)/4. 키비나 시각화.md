## 키비나 시각화

#### 1. 인덱스 매핑
- ★`logstash` => 보통 로그는 날짜를 인덱스로 저장한다 

```
#18~20까지 세개의 인덱스 구성 필요 
PUT logstash-2015.05.18
{
  "mappings": {
    "properties": {
      "geo": {
        "properties": {
        "coordinates": { "type": "geo_point" } 
        } 
      }
    } 
  } 
}

PUT logstash-2015.05.19
{
  "mappings": {
    "properties": {
      "geo": {
        "properties": {
        "coordinates": { "type": "geo_point" } 
        } 
      }
    } 
  } 
}

PUT /logstash-2015.05.20
{
  "mappings": {
    "properties": {
      "geo": {
        "properties": {
        "coordinates": { "type": "geo_point" } 
        } 
      }
    } 
  } 
}
```
